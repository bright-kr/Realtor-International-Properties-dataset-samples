## Data Points

| Field Name | Description |
|------------|-------------|
| `url` | Realtor에서 해당 부동산의 상세 매물 페이지로 연결되는 직접 URL입니다. |
| `id` | 각 부동산 매물에 할당된 고유 식별자입니다. |
| `title` | 주소 세부 정보를 포함한 부동산 매물의 제목 또는 헤드라인입니다. |
| `price` | 지정된 통화로 표시된 부동산의 등록 가격입니다. |
| `currency` | 부동산 가격이 표시되는 통화입니다. |
| `description` | 부동산의 특징 및 편의시설에 대한 간단한 설명 또는 개요입니다. |
| `images` | 목록 형태로 제공되는, 부동산을 보여주는 이미지의 URL입니다. |
| `bedrooms` | 해당 부동산에서 이용 가능한 침실 수입니다. |
| `bathrooms` | 해당 부동산에서 이용 가능한 욕실 수입니다. |
| `property_type` | 부동산 유형(예: house, apartment, villa)입니다. |
| `building_size` | 제곱피트 또는 제곱미터 기준의 건물 총면적입니다. |
| `land_size` | 해당 부동산이 지어진 토지의 총면적입니다. |
| `rooms` | 해당 부동산에서 이용 가능한 전체 방 수입니다. |
| `address` | 부동산의 전체 도로명 주소입니다. |
| `country_code` | 부동산이 위치한 국가를 나타내는 ISO 코드입니다. |
| `state` | 부동산이 위치한 주 또는 지역입니다. |
| `city` | 부동산이 위치한 도시입니다. |
| `realtor_details` | 해당 부동산 매물을 관리하는 realtor 또는 agency에 대한 정보입니다. |
| `price_history` | 시간 경과에 따른 부동산 가격 변동의 이력 데이터입니다. |
| `listing_status` | 매물의 현재 상태(예: active, pending, sold)입니다. |


## Use Cases

### 1. Market Analysis
이 데이터셋을 활용하여 다양한 지역 전반의 트렌드와 가격 책정 전략을 파악하는 등 포괄적인 시장 분석을 수행할 수 있습니다.

### 2. Investment Opportunities
부동산 세부 정보와 과거 가격 데이터를 분석하여 수익성 높은 부동산 투자 기회를 식별할 수 있습니다.

### 3. Competitive Benchmarking
데이터셋을 사용하여 부동산 매물을 비교하고 부동산 시장에서의 경쟁 포지셔닝을 평가할 수 있습니다.
